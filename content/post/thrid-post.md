## HTML Basics

HTML (HyperText Markup Language) is the backbone of any website. It provides the structure of your site and is used to organize and present content.

### Elements and Tags

HTML consists of elements and tags. For example:

```html
<!DOCTYPE html>
<html>
  <head>
    <title>My Website</title>
  </head>
  <body>
    <h1>Welcome to My Website</h1>
    <p>This is a sample paragraph.</p>
  </body>
</html>
```

```


### Forms and Inputs

Forms are essential for user interaction. Here's an example:

```html
<form action="/submit" method="POST">
  <label for="name">Name:</label>
  <input type="text" id="name" name="name" required />
  <button type="submit">Submit</button>
</form>
